# 🚀 API de Facturación - Python & FastAPI

Esta es una API de facturación desarrollada con **Python** y **FastAPI**, utilizando **MySQL** como motor de base de datos. El proyecto implementa una estructura de base de datos normalizada para gestionar clientes, productos y facturas de manera eficiente.

## 🛠️ Tecnologías utilizadas

* **Lenguaje:** [Python](https://www.python.org/)
* **Framework:** [FastAPI](https://fastapi.tiangolo.com/)
* **Base de Datos:** MySQL (gestionado con XAMPP)
* **Servidor ASGI:** Uvicorn

## 📋 Características

* Gestión de inventario (Productos).
* Registro y administración de Clientes.
* Generación de facturas con relaciones normalizadas.
* Documentación interactiva automática con Swagger UI.
